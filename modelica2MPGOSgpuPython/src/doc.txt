  when sample(...) then

MAPPING VariableCategory |-> MPGOS type
        
        state -------------> X
        derivative --------> F (deve avere lo stesso id di X se hanno lo stesso QualifiedNamePart)
        algebraic ---------> ACC  (Real), ACCi  (Integer)
        ind.Param/Const ---> sPAR (Real), sPARi (Integer)
